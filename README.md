## Observación
La primera vez que se instala la app, al realizar una búsqueda no se muestra ningún resultado (sin importar cuántas búsquedas se realicen), pero reiniciándola se arregla y no vuelve a pasar a menos que se desinstale y vuelva a instalar.  

Según me comentaron otros compañeros, los alojamientos iniciales no se cargan en la base de datos hasta realizar una consulta, por lo que probé realizar una tanto al crear la base de datos (desde la clase AppDataBase) como al crear la MainActivity, pero seguía ocurriendo lo mismo.  

Como no pude encontrar el error y se puede resolver simplemente reiniciando la app luego de la instalación, lo termine dejando como estaba.
